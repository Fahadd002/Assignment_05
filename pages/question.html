Question-1: What are the different ways to select an element in the DOM?

getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets/sets HTML content inside an element.
innerText: Gets/sets the visible text inside an element.
textContent: Gets/sets all text inside an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event listeners.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the innermost element and propagates outward to the parent elements.
